// CYBERSECURITY FIELD MANUAL

Essential threat intelligence and protection protocols — curated by CyberShield AI

🔑

PASSWORD SECURITY

→
Use minimum 12 characters with mixed case, numbers and symbols.
→
Never reuse passwords across multiple platforms or services.
→
Use Bitwarden — a free, open-source password manager.
→
Enable 2FA on every critical account without exception.
🔗

SAFE BROWSING

→
Always verify URL authenticity before entering credentials.
→
HTTPS + padlock = encrypted. HTTP = never use for sensitive data.
→
Type URLs manually — never click links in emails.
→
Only download software from official, verified sources.
📱

MOBILE SECURITY

→
Install apps exclusively from Play Store or App Store.
→
Audit app permissions — deny unnecessary access requests.
→
Keep OS and apps updated — patches fix exploits.
→
Use biometric lock — never share your PIN.
📧

PHISHING DEFENSE

→
Banks never request passwords or OTPs via email or SMS.
→
Urgent prize or threat messages = immediate red flag.
→
Never share Aadhaar, PAN or bank details via any chat.
→
Verify suspicious contact by calling the official number.
💻

DEVICE HARDENING

→
Apply OS updates immediately — they patch critical CVEs.
→
Backup data — 3-2-1 rule: 3 copies, 2 media, 1 offsite.
→
Lock screen when leaving workstation unattended.
→
Use Windows Defender or reputable AV software.
🌐

SOCIAL ENGINEERING

→
Never expose address, phone or government IDs online.
→
Set all social profiles to private — limit visibility.
→
Unknown contacts requesting money = social engineering.
→
Always log out of accounts on shared devices.
